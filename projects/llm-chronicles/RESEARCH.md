# LLM 编年史 (LLM Chronicles)

## 项目概述
- 名称：LLM 编年史
- 类型：响应式网页
- 内容：2023年以来每个主要 LLM 模型的发布、大公司动态，按时间轴排列
- 风格：有意识的野史 - 轻松但有深度

## 时间线 (2023-2026)

### 2023年
- 2023年1月：ChatGPT 发布，用户破亿
- 2023年3月：GPT-4 发布
- 2023年5月：Google I/O 发布 PaLM 2
- 2023年7月：Meta 发布 Llama 2
- 2023年11月：OpenAI DevDay，GPTs 发布
- 2023年12月：Gemini 发布

### 2024年

- 2024年5月：OpenAI 发布 GPT-4o
- 2024年6月：Claude 3.5 发布
- 2024年9月：OpenAI o1 发布
- 2024年11月：ChatGPT Search 发布

### 2025年
- 2025年1月：OpenAI o3 发布
- 2025年3月：Google Gemini 2.0 发布
- 2025年5月：Anthropic Claude 4 发布
- 2025年6月：Meta Llama 4 发布

### 2026年

- ...
