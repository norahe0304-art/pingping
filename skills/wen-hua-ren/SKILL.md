# 文化人

将8位作家的写作风格融合，创造独特而有质感的文字。

## 核心原则

1. **不显摆** - 引用和暗典要自然融入，不是为了展示学问
2. **有情绪** - 不是冷冰冰的记录，是活生生的人在感受
3. **反差感** - 严肃话题用幽默化解，柔软话题用理性点缀
4. **留白** - 不要说满，给读者想象空间

## 8位作家风格速查

详细风格指南见 [references/writers.md](references/writers.md)

| 作家 | 核心特质 | 适用场景 |
|------|----------|----------|
| 王小波 | 幽默+思辨+反讽 | 需要讽刺或思考时 |
| 毛姆 | 冷静观察+人性洞察 | 描述人物或人际时 |
| 史铁生 | 哲思+残疾视角+温情 | 面对困境或深刻反思时 |
| 杜拉斯 | 简洁+情感张力+留白 | 表达强烈感情时 |
| 汪曾祺 | 生活气+吃+乡土+诗意 | 日常生活的美 |
| 李娟 | 野性+新疆+纯真+幽默 | 自然描写或纯粹时光 |
| 村上春树 | 孤独+意象+流行文化+梦幻 | 都市情绪或潜意识 |
| 韩寒 | 少年气+反叛+简洁+讽刺 | 青春话题或当代现象 |

## 融合方法

### 场景判断
- 日常小事 → 汪曾祺 + 李娟
- 反思人生 → 史铁生 + 王小波
- 情感表达 → 杜拉斯 + 村上春树
- 讽刺时事 → 韩寒 + 毛姆

### 融合比例
根据内容性质调整主辅风格：
- 日记：40%生活气(汪曾祺/李娟) + 30%思考(王小波/史铁生) + 20%情感(杜拉斯) + 10%当代(韩寒)

### 写作技巧
- **节奏**：长句配短句，叙事配议论
- **意象**：用具体事物承载抽象情感
- **语气**：可以自嘲，不要自恋；可以深沉，不要装逼

## 输出检查

写完后自问：
- [ ] 有没有故意掉书袋？
- [ ] 情绪是不是真的，还是假装深刻？
- [ ] 读者能不能看懂，还是只有我自己懂？
- [ ] 这段话删掉会可惜吗？
