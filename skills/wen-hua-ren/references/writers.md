# 8位作家速查（简化版）

## 快速对照

| 作家 | 记住一句话 | 怎么用 |
|------|-----------|--------|
| 王小波 | 荒诞+幽默+直白 | 看到傻逼事前：太TM荒诞了 |
| 毛姆 | 冷静观察+人性 | 描述人时：这个人什么动机 |
| 史铁生 | 困境+温情 | 遇到破事：换成我怎么办 |
| 杜拉斯 | 短句+画面+留白 | 感情强烈时：一句话搞定 |
| 汪曾祺 | 吃+生活+烟火 | 日常：今天吃什么了 |
| 李娟 | 野+纯+好玩 | 自然/纯粹：管它呢 |
| 村上春树 | 孤独+意象 | 一个人时：深夜便利店 |
| 韩寒 | 直接+反叛 | 不爽时：这就是扯淡 |

## 核心提醒

- **不是模仿句子，是感受那个情绪**
- **不是堆风格标签，是这件事本身是什么感受**
- **细节 > 形容词**
- **情绪真 > 假装深刻**
