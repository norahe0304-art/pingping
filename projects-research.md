# 三个方向深度研究

---

## 项目1：自动化外贸 Outbound

**描述**：用 Apollo/Hunter 找 leads，AI 生成个性化 cold email，帮客户约会议  
**收费模式**：按会议数或月费收费  
**状态**：方案阶段

### 目标客户画像

- **主要**：中小型外贸公司（10-50人），没有专职 BD
- **次要**：国际销售团队，想 scalable 地做 cold outreach
- **付费意愿**：愿意为"省时间+有效果"付费
- **痛点**：手动找联系人太慢，邮件回复率低，约不到会议

### 竞品分析

| 竞品 | 定位 | 定价 | 弱点 |
|------|------|------|------|
| Apollo.io | 全栈 B2B 数据+自动化 | $39-79/月 | 贵，个性化一般 |
| Hunter.io | 邮箱查找+验证 | $49/月起 | 只管邮箱 |
| Instantly | Cold email 自动化 | $29/月起 | 数据源弱 |
| Copy.ai | AI GTM 平台 | $49/月起 | 非专注 outreach |

### 技术方案

```
数据层：Apollo API / Hunter API → 获取联系人
AI 层：GPT-4 → 生成个性化邮件（读取 LinkedIn/公司信息）
发送层：Amazon SES / SendGrid → 批量发送
预约层：Calendly API / Cal.com → 会议预约
自动化：n8n / Zapier → 工作流编排
```

### 定价策略

- **方案 A**：$99/月，无限邮件 + 基础 AI
- **方案 B**：$50/个预约成功的会议
- **方案 C**：$299/月，高级 AI + CRM 集成

### MVP 验证

1. 先用 Apollo + ChatGPT 手动跑通 1-2 个客户案例
2. 确认流程：找联系人 → 生成邮件 → 发送 → 跟进 → 预约
3. 定价测试：先收低价，成功后再涨价

## 项目2：批量广告素材自动化

**描述**：给品牌批量出社媒图+文案+多平台适配，3天交付几百组素材  
**Demo 案例**：biuty.ai（AI 护肤 app）  
**状态**：Demo 阶段（暂停）

### 目标客户

- **主要**：DTC 品牌、电商卖家（预算有限，请不起设计师）
- **次要**：MCN / 代运营公司（需要快速出大量素材）
- **付费意愿**：按生成量付费，便宜好用就大量用

### 交付流程

```
1. 客户上传产品图 + 描述
2. AI 生成多版本素材（不同风格/文案）
3. 人工审核+调整（可选）
4. 输出多平台适配版本（小红书、IG、FB、TikTok）
5. 交付
```

### 定价模式

- **按张**：$0.1-0.3/张
- **按套餐**：$99/月，500张；$199/月，1500张
- **按项目**：$500-2000/品牌活动

### 为什么暂停？

可能原因：
- 交付质量不稳定（AI 生成的图有时不像产品）
- 人工审核成本高
- 定价太低，利润薄
- 客户教育成本高（需要解释 AI 能做什么）

### 重新启动建议

1. **聚焦单一渠道**：先只做小红书，做精
2. **先做爆款模板**：不要让客户自己描述，给他们选"爆款风格"
3. **提价**：$99/月的价格太低，至少 $199
4. **先收钱再做事**：不做免费 demo

## 项目3：SEO 自动化

**描述**：接 GA4+GSC 数据，自动分析关键词机会，批量生成优化内容推到 CMS  
**收费模式**：按月订阅收费  
**状态**：方案阶段

### 目标客户

- **主要**：中小企业的独立站（没有专职 SEO）
- **次要**：WordPress/WooCommerce 站长
- **付费意愿**：$49-199/月，愿意为"省时间"付费
- **痛点**：不懂 SEO，内容写不出来，写了也不优化

### 竞品分析

| 竞品 | 定位 | 定价 | 弱点 |
|------|------|------|------|
| Surfer SEO | AI 内容优化 | $69/月起 | 只管内容，不管发布 |
| Jasper | AI 写作 | $49/月起 | 非专注 SEO |
| Frase.io | SEO Briefs | $15/月起 | 功能基础 |
| Clearscope | 企业内容优化 | $170/月起 | 贵，非中小企业 |

### 技术方案

```
数据层：Google Search Console API + GA4 Data API → 获取搜索数据
分析层：关键词聚类 + 机会分析 → 输出内容主题
生成层：GPT-4 → 生成优化过的内容
发布层：WordPress REST API / Webflow API → 推送到 CMS
监控层：GSC → 追踪排名变化
```

### 定价策略

- **方案 A**：$49/月，10 页优化 + 发布
- **方案 B**：$99/月，30 页优化 + 发布
- **方案 C**：$199/月，无限页 + 高级分析

### MVP 验证

1. 先手动跑 1-2 个网站的 SEO 优化（用 GSC 数据指导内容）
2. 确认效果：3-6 个月看排名变化
3. 工具化：把手动流程写成自动化脚本
4. 定价测试

---

## 优先级建议

如果只能选一个先做：

1. **SEO 自动化** — 需求明确，技术可行，竞品教育过市场
2. **Outbound** — 市场需求大，但竞争激烈，需要差异化
3. **素材自动化** — 看起来简单，但做好很难，暂停原因需确认
