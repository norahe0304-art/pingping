# OpenTwitter MCP: Twitter数据的AI查询服务

## 内容摘要

1. **核心功能**：让 AI Agent 能直接查询 Twitter/X 数据的 MCP 服务
2. **用户资料查询**：获取账号的详细资料信息
3. **推文搜索**：通过关键词搜索推文
4. **粉丝事件追踪**：追踪粉丝变化事件
5. **KOL 追踪**：关键意见领袖动态追踪
6. **结构化数据**：API 返回结构化数据，包括粉丝数、关注数、发推数、是否认证等
7. **内容调研价值**：想了解某个 KOL 的背景，不需要手动刷 Twitter，直接让 Agent 查询就能拿到结构化信息
8. **舆情监控适用**：搜关键词、追踪特定账号动态都很合适
9. **API 配额注意**：可能需要 Twitter API 配额，fxtwitter API 是免费替代方案

## 我们可以做什么

1. **Twitter 舆情监控**：利用 MCP 搜特定关键词、追踪 KOL 动态
2. **做内容调研**：快速了解某个账号的背景信息，作为内容创作素材
3. **学习 MCP 写法**：研究如何写一个 MCP server，作为开发参考
4. **方案对比**：对比 OpenTwitter MCP 和 fxtwitter API 的功能差异，根据需求选择
