# OpenRouter统一API - 解读

## 核心价值
- 一个API key调用400+模型
- 自动降级，某提供商挂了自动切换
- 5.5%手续费，但推理价格和官方一致

## 关键洞察
- 模型 landscape 每周都在变
- 锁定单一提供商是赌注
- OpenRouter让你用"一句话"切换模型
