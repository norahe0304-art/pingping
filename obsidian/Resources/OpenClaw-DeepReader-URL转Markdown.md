# DeepReader - 任意URL自动转Markdown

## 内容摘要

1. **核心定位**：OpenClaw 内置的内容读取器，把任意 URL 自动抓取、清洗、保存为高质量 Markdown
2. **零配置零 API**：不需要任何配置和 API Key，直接可用
3. **支持的来源广泛**：X (Twitter) 帖子和线程、Reddit 帖子、YouTube 字幕、任意网页内容
4. **自动清洗功能**：去除 HTML 广告和噪音，保存干净的 Markdown 格式
5. **对资源沉淀工作流有用**：DeepReader 负责抓取，Agent 负责理解沉淀，整个流程可自动化
6. **免费无成本**：不需要 API Key，对"能省则省"的团队是加分项
7. **去广告效果好**：测试发现比直接爬虫好很多，尤其在去除广告和噪音方面
8. **可配合 cron 使用**：支持定时抓取特定来源的更新

## 我们可以做什么

1. **配置 DeepReader skill**：在 OpenClaw 中启用 DeepReader
2. **建立自动化抓取流程**：设置几个重点来源的定时抓取
3. **测试抓取质量**：针对不同来源测试清洗效果，优化规则
4. **实现全自动化沉淀**：资源频道链接→自动抓取→自动沉淀形成闭环
