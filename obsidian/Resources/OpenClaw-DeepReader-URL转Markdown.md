# DeepReader: 任意URL自动转Markdown

## 内容摘要

1. DeepReader 是 OpenClaw 内置的内容读取器
2. 核心功能：把任意 URL 自动抓取、清洗、保存为高质量 Markdown
3. 零配置、零 API Key，直接可用
4. 支持读取 X (Twitter) 帖子和线程
5. 支持读取 Reddit 帖子
6. 支持读取 YouTube 字幕
7. 支持读取任意网页内容
8. 自动清洗 HTML，去除广告和噪音
9. 保存为 Markdown 格式，便于后续处理
10. 与"资源沉淀"工作流高度契合：DeepReader 负责抓取，Agent 负责理解沉淀

## 我们可以做什么

1. **自动化抓取**：配合 cron，定时抓取特定来源的更新
2. **每日资讯自动抓取**：建立自动化流程，每天自动抓取行业资讯
3. **内容清洗**：利用自动清洗功能，直接获得干净的内容用于分析
4. **全自动化沉淀**：看到资源频道的链接→自动抓取→自动沉淀，形成闭环
