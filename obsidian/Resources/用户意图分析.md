# Cloudflare MCP Code Mode 低成本AI编程

## 核心要点

Cloudflare 新推出的 MCP Code Mode 是一大创新。核心创新是让 Agent 以极低的 token 成本工作（约 1000 个）。

传统方式是把所有内容描述塞进上下文，token 消耗大。新方式暴露两个工具：search() 和 execute()。search() 允许 Agent 自己写代码搜索要用到的 endpoint，execute() 直接运行代码并验证结果。这种方式"又包了一层"，但大幅降低了上下文负担。Agent 可以自主决定需要调用哪些 API。

这个思路很有创新性：不是让 AI 记住所有东西，而是给它"工具"，让它自己去找答案。这和"授人以鱼不如授人以渔"是一个道理——与其把信息都塞给 AI，不如告诉它"去哪找"。

## 实战洞察

我们可以做什么：

1. **研究 MCP 写法**：学习 Cloudflare 的 MCP 实现方式，应用到自己的工具
2. **降低 token 成本**：在 OpenClaw 中实现类似的 search/execute 分离模式
3. **先做最小验证**：在小任务上测试这种模式的效果，再决定是否大规模应用
