# 方正证券OpenClaw金融投研应用系列报告

## 内容摘要

1. **报告来源**：方正证券研究所发布的 OpenClaw 在金融投研领域的研究报告
2. **报告篇数**：共三篇，第二篇是部署指南 + 基础金融功能测试，第三篇是 17 个高效应用案例详解
3. **技术架构**：Gateway（网关）+ Agent（智能体）+ Skills（技能工具）+ Memory（记忆系统）
4. **核心优势**：实现从"对话"到"做事"的跨越
5. **部署策略**：隔离部署，用云服务器而非本地电脑，推荐腾讯云 Lighthouse 一键部署
6. **安全原则**：最小权限 + 隔离环境，降低数据泄露风险
7. **17 个案例分类**：基础效率类（5个）、内容生成类（4个）、金融投研类（7个）
8. **金融投研案例**：同花顺/米筐/Wind API 接入、PB-ROE 选股策略、杯柄形态选股、股票深度分析、全自动因子挖掘与回测
9. **风险提示**：历史规律可能失效、AI 大模型存在幻觉、案例仅供测试不构成投资建议
10. **对 LinkCard 价值**：投研场景复用、agentcard 参考、"从对话到做事"的产品设计思路

## 我们可以做什么

1. **参考部署方案**：使用腾讯云 Lighthouse 一键部署方案进行测试
2. **复用金融 Skills**：API 数据提取、报告撰写、策略复现等金融 Skills 可直接借鉴
3. **产品设计参考**：参考"从对话到做事"的产品设计思路
4. **案例集整理**：把 17 个案例整理成"Agent 做什么"案例集
5. **注意风险**：涉及金融操作时保持谨慎，AI 可能有幻觉
