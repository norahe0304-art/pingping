# OpenNews MCP: 加密货币新闻聚合与AI评分

## 内容摘要

1. **核心功能**：加密货币新闻聚合工具，聚合多个新闻来源并用 AI 评分系统筛选
2. **AI 评分系统**：给新闻打上 Safe / Risky / Problematic 标签，并给出评分理由
3. **交易信号**：提供交易信号，辅助投资决策
4. **实时更新**：支持新闻实时更新
5. **质量判断价值**：不只是简单聚合新闻，而是用 AI 给出"质量判断"，比单纯摘要更有价值
6. **团队背景**：来自 6551 团队，他们同时也在做 MCP 平台
7. **架构参考价值**：对于非加密货币方向的我们，这个项目的 MCP 写法比直接使用更有参考价值
8. **垂直领域新闻聚合**：可作为以后做垂直领域新闻聚合的架构参考

## 我们可以做什么

1. **学习 MCP 写法**：研究这个 MCP server 的实现方式，作为开发参考
2. **参考新闻聚合模式**：以后做垂直领域（AI/科技）新闻聚合时可借鉴其架构
3. **评估整合价值**：如果做加密货币相关的内容，这个工具可能有整合价值
4. **技术储备**：目前对日常运营价值有限，但可作为技术储备
