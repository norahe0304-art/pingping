# PARA 三分区知识流框架

## 核心方法

- **Resources**: 资源频道 URL 的深度阅读总结（知识编译层）
- **Areas**: 从 Resources 和聊天模式中提炼的可复用 SOP/框架（复用层）
- **Social**: 需要发布或待发布的内容资产（执行层）

## 执行要点

1. 新内容先进入 Resources 加工
2. 重复出现 >=2 次的模式 → 提炼到 Areas
3. 明确要发布的内容 → 放入 Social
