## 核心原则

### 目标用户画像
- 大脑简单，不想动脑
- 要快速获得价值
- 商业场景用户
- 非技术人员

### 成功案例特征
1. 使用场景简单：一个具体问题
2. 高效：一下就搞定
3. 懒：不想自己动手

### 内容方向（可直接用）
- 文档自动整理
- 批量图片处理
- 自动生成报告
- 客户资料提取
- 会议纪要

### 爆款逻辑
- 发出去有面子
- 别人会来问"你怎么做到的"
- 转发给朋友看

### 典型案例
- brand visual → markdown
- 终端特效（cmatrix、hollywood）
- 桌面美化
- 电子宠物
- 炫酷 GitHub README
